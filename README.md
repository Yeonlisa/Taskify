# Taskify
ToDo-List에 `react-beautiful-dnd` 라이브러리를 사용해 Drag & Drop 기능까지 추가해보았다.

## 빌드 및 실행방법
root directory에서 `npm install` 한뒤 `npm run start`한다.<br>
추가 배포시 `npm run deploy`를 한다.
